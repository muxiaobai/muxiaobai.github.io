GitHub + Hexo 构建我的博客

有了Hexo后，我的文章就准备全部迁移至Github了。

1.本机搭建

使用环境

git node npm

Github账户 username.github.io 仓库

安装Hexo

Hexo安装，要用全局安装，加-g参数。

查看hexo的版本

hexo 帮助命令

命令行解释：

  • help 查看帮助信息
  • init 创建一个hexo项目
  • migrate 从其他系统向hexo迁移
  • version 查看hexo的版本
  • –config参数，指定配置文件，代替默认的_config.yml
  • –debug参数，调试模式，输出所有日志信息
  • –safe参数，安全模式，禁用所有的插件和脚本
  • –silent参数，无日志输出模式

安装好后，我们就可以使用Hexo创建项目了。
